Системы контроля версий.

С помощью систем контроля версий возможо скачать самую последнюю версию проекта либо какую-то из предыдущих версий. Перед тем как начинать работу с проектом необходимо скачать последнюю версию, сделать изменения и затем сделать коммит (сохранить изменения) и пуш (отправить изменения в удалённый репозиторий).

Последовательность операций:

  1. получить (pull, update) — взять последнюю версию кода проекта.
  2. сделать правки в коде проекта.
  3. закомитить (commit) — утвердить изменения: написать комментарий к изменениям, выделить файлы которые меняются.
  4. отправить (push) — отправить изменения в репозиторий.

Компонент Tooltip.

Компонент Tooltip представляет собой всплывающую подсказку появляющуюся при наведении на любой элемент веб-страницы.

С технической точки зрения Tooltip состоит из двух частей:

  • элемент вызывающий появление всплывающей подсказки
  • всплывающая подсказка

Если всплывающая подсказка представляет собой просто текст, реализовать её можно добавив дополнительный атрибут внутри которого будет размещаться текст всплывающей подсказки. Стилями при наведении на элемент добавляются псевдоэлементы :before и :after в которых содержится сам текст с дизайном и стрелочка.

See the Pen Pure CSS tooltip by Aleksey Korovin (@alekskorovin) on CodePen.

Немного меняем стили и Tooltip отображается над текстом:

See the Pen Pure CSS tooltip above by Aleksey Korovin (@alekskorovin) on CodePen.

Недостатком такого варианта является фиксированная позиция всплывающей подсказки которая не зависит от положения курсора мыши, находящегося поверх какой-то части элемента.

В случае если необходима подсказка которая будет спозиционирована относительно элемента и в элементе кроме текста может быть HTML, используются специализированные JavaScript-библиотеки, позволяющие подключить функциональность Tooltip-а к элементам с помощью JavaScript.

Компонент Dropdown.

Стандартный элемент HTML-формы <select> обладает не очень хорошими возможностями для его стилизации.

See the Pen select by Aleksey Korovin (@alekskorovin) on CodePen.

В случае нестандартного дизайна этого элемента необходимо использовать JavaScript для скрытия элемента select и дорисовывания новых элементов.

Плагин Select2 (https://select2.github.io/examples.html) это много-функциональное решение и подходит если в проекте необходимо менять дизайн в том числе и выпадающих элементов либо нужна функциональность добавления нескольких значений в одно поле либо другая сложная функциональность.

В случае если дизайн показывает только одно состояние элемента, мы можем обойтись минимальными скриптами и стилями. Все что нам надо это отображать текущее состояние элемента с выбранным значением и дорисовывать стрелочку, при этом мы можем с помощью CSS изменить внешний вид самого элемента, а так же самой стрелки.

Задание #3.1

  2. Скачать PSD файл mobi-contact-desktop-dropdown.psd и открыть его в Photoshop-е.
  3. Открыть в редакторе кода HTML-файл mobi/contact.html и перед закрывающим тегом body разместить следующий JavaScript-код: 
    <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.1.1/jquery.min.js"></script>
<script>
    $(document).ready(function() {

        $.fn.customSelect = function() {
            $(this).wrap('<div class="select-wrapper">');
            var setActiveElement = function (elem) {
                var $elem = $(elem),
                $option = $elem.find('option:selected');

                $elem.parent().attr('data-select-label', $option.text());
            };

            setActiveElement($(this));

            $(this).on('change', function () {
                setActiveElement($(this));
                $(this).blur();
            });
        };

        $('select').each(function() {
            $(this).customSelect();
        });
    });
</script>

    overflow: hidden;

    padding: 0 0.4rem;
    display: block;
    height: 3rem;

    content: attr(data-select-label);
    line-height: 2;

.select-wrapper:hover {
    background: #ccc;

    content: '';
    position: absolute;
    right: 0.5rem;

    width: 0;

    left: 0;
    top: 0;
    position: absolute;

    height: 100%;
    opacity: 0;

/* Custom select */
.contact-us-c .select-wrapper {

Резиновая вёрстка.

Общепринято различать четыре основных вида вёрстки:

  • Фиксированная вёрстка (fixed layout) — когда ширина основных элементов каркаса сайта задана в пикселях либо не относительных единицах.

    See the Pen Floating elements grids - fixed by Aleksey Korovin (@alekskorovin) on CodePen.

  • Резиновая вёрстка (liquid layout) — ширина колонок и элементов выстраивающихся по горизонтали задана в процентах.

    See the Pen Floating elements grids - liquid by Aleksey Korovin (@alekskorovin) on CodePen.

  • Адаптивная вёрстка (adaptive layout) — когда ширина колонок после определённой ширины экрана скачкообразно меняется на фиксированное значение.

    See the Pen Floating elements grids - adaptive by Aleksey Korovin (@alekskorovin) on CodePen.

  • Отзывчивая вёрстка (responsive layout) — когда ширина колонок после определённой ширины экрана меняется на ширину в относительных единицах.

    See the Pen Floating elements grids - responsive by Aleksey Korovin (@alekskorovin) on CodePen.

Если уменьшить ширину экрана текущей страницы contact.html — к какому типу относится такая вёрстка?

Задание #4.1

  1. Страница contact.html сделана с элементами резиновой и фиксированной вёрстки, необходимо привести её к варианту резиновой вёрстки.
  2. Нам надо убрать некоторые недочёты. Открыть в редакторе кода mobi/css/main.css и заменить фиксированную высоту на минимальную высоту: 
    .contact-us {
    margin: 0 auto;
    height: 440px;
    background-color: #44556c;
}
    поменять на: 
    .contact-us {
    margin: 0 auto;
    min-height: 440px;
    background-color: #44556c;
}
    Теперь элемент "падающий" под другой элемент будет виден.
  3. Чтобы флоатнутые элементы не падали при уменьшении размера экрана, надо задать их ширину в процентах: 
    .clobal {
    width: 483px;
    float: left;
    padding-top: 83px;
}
    заменим на: 
    .clobal {
    width: 50%;
    float: left;
    padding-top: 83px;
}
  4. То же самое проделать и для второго блока (так же заменим фиксированную высоту на минимальную): 
    .form_c {
    width: 484px;
    height: 440px;
    padding-top: 23px;
    float: left;
}
    заменить на: 
    .form_c {
    width: 50%;
    min-height: 440px;
    padding-top: 23px;
    float: left;
}
  5. Поменять стили для элементов форм — заменить значения в px на значения в процентах.
  6. Обратить внимание что остальные блоки изначально резиновые.
